Buat Kartu NISN

Untuk membuat kartu NISN hanya untuk 1 data, bisa menuju ke halaman ini -->

  1. Unduh template Excel di sini -->
  2. Edit dan isi template Excel sesuai dengan data pada DAPODIK
  3. Jangan mengubah nama kolom pada template Excel!
  4. Nama yang terlalu panjang dapat diatasi dengan menyingkatnya agar tidak melewati batas kartu
  5. Ubah warna QR-Code sesuai dengan keinginan Anda
  6. Tekan "Upload File Excel" di bawah ini jika sudah selesai mengisi data, kemudian pilih template Excel yang sudah diisi data
  7. Tekan tombol Proses
  8. Gulir ke bawah untuk melihat hasil
  9. Tekan tombol Cetak
  10. Untuk mengulangi dari awal, tekan tombol Reset
