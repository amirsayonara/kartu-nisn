Buat Kartu NISN Single

Untuk membuat kartu NISN dengan data yang banyak, bisa menuju ke halaman ini -->

  1. Lengkapi data pada kolom isian di bawah ini
  2. Nama yang terlalu panjang dapat diatasi dengan menyingkatnya agar tidak melewati batas kartu
  3. Ubah warna QR-Code sesuai dengan keinginan Anda
  4. Tekan tombol Buat jika data sudah lengkap
  5. Gulir ke bawah untuk melihat hasil
  6. Tekan tombol Cetak
  7. Untuk mengulangi dari awal, tekan tombol Reset
